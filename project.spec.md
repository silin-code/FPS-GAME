# 恐怖生存 FPS — 项目 SPEC

> 版本: v0.1 (访谈第一轮)
> 目标: 1–2 个月内做出可玩原型

---

## 1. 游戏概览

| 项目 | 内容 |
|------|------|
| **游戏名称** | (待定) |
| **类型** | 恐怖生存 FPS |
| **目标平台** | Windows PC |
| **视角** | 第一人称 / 第三人称 可切换 |
| **核心玩法** | 单人剧情 + 探索解谜 + Boss 战 |

---

## 2. 故事背景

- **设定**: 废弃小镇（Resident Evil / 寂静岭 风格）
- **氛围**: 压抑、昏暗、未知恐惧
- **玩家角色**: 误入小镇的主角，寻找逃生之路
- **剧情基调**: 通过环境叙事 + 少量文档/录音，讲述小镇发生了什么

---

## 3. 核心系统

### 3.1 角色控制

| 功能 | 描述 |
|------|------|
| 移动 | WASD 移动，Shift 跑步，Ctrl 下蹲，空格跳跃 |
| 视角切换 | 按 V 键在第一人称/第三人称间切换 |
| 瞄准 | 右键瞄准 (ADS) |
| 交互 | E 键拾取/开门/操作物品 |

### 3.2 战斗系统 (中等节奏)

- 武器: 1–2 把 (手枪 + 霰弹枪 或 手枪 + 步枪)
- 弹药有限但不极度稀缺，需要管理
- 换弹: R 键
- 射击: 左键

### 3.3 物品/资源管理

- 拾取: 弹药、血包、钥匙/道具
- HUD: 血量、弹药数、道具快捷栏

### 3.4 存档系统

- 手动存档点 (打字机/收音机 等场景道具)
- 每次存档记录玩家状态和世界状态

---

## 4. 关卡设计

### 4.1 整体结构

```
废弃小镇入口 (新手引导)
    ↓
街区探索 (收集资源 + 解谜)
    ↓
教堂/市政厅 (解谜核心区域)
    ↓
地下通道 (最终区域)
    ↓
Boss 战 → 逃脱
```

### 4.2 关卡要素

| 要素 | 说明 |
|------|------|
| 探索 | 开放式街区，有多条路径可达目标 |
| 解谜 | 2–3 个轻度解谜 (找钥匙、拼图、通电等) |
| 战斗 | 遭遇小型敌人 (如丧尸/怪物) + 最终 Boss |
| 氛围 | 动态灯光、音效、雾气 |

---

## 5. Boss 设计

- **Boss 类型**: 大型变异怪物
- **阶段**: 2 阶段
  - 阶段 1: 追逐玩家，弱点在身体特定位置
  - 阶段 2: 狂暴化，攻击模式变化
- **战斗场地**: 地下空间 / 教堂大厅，有遮挡物
- **击杀条件**: 击中弱点 + 场景互动 (如引爆油桶/发电机)

## 6. 技术栈

| 技术 | 选择 |
|------|------|
| 引擎 | Unity (推荐 2022 LTS 或 6000.0 LTS) |
| 语言 | C# |
| 渲染管线 | URP (Universal Render Pipeline) |
| 输入系统 | Unity Input System 包 |
| 动画 | Unity Animator + Cinemachine |
| 音效 | Unity Audio |
| 版本控制 | Git + GitHub |

---

## 7. 资源策略

1. **主角模型**: Unity 官方的 Third Person / First Person 角色包 (免费)
2. **环境模型**: Unity Free Low Poly / Urban 包 或 Asset Store 免费资源
3. **武器模型**: Unity FPS Microgame 资源包
4. **敌人/Boss**: 使用少量付费/免费资源，或自己 Blender 简单建模
5. **UI**: 使用 Unity UI 系统，自己制作
