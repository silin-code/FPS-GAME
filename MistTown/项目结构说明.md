# 《迷雾镇》项目结构说明

> 本文档解释整个 Unity 项目的文件夹是干什么的。

---

## 根目录 (MistTown/)

这是 Unity 项目的根文件夹。

| 文件/文件夹 | 说明 |
|------------|------|
| `Assets/` | **你的主战场** — 所有游戏资源（代码、模型、场景、声音）都放这里 |
| `Packages/` | Unity 包管理文件，一般不用动 |
| `ProjectSettings/` | Unity 编辑器设置，一般不用动 |
| `Library/` | Unity 自动生成的缓存，**不用管** |
| `Temp/` | 临时文件，**不用管** |
| `obj/` | 编译中间文件，**不用管** |
| `Logs/` | Unity 日志，**不用管** |
| `UserSettings/` | 编辑器个人设置，**不用管** |
| `MistTown.sln` | Visual Studio 解决方案文件，双击用 VS 打开项目 |
| `.vsconfig` | VS 组件配置 |
| `项目结构说明.md` | **就是你现在看的这个文件** |

## Assets/ 详解

### `Assets/_Scripts/` — C# 脚本

所有游戏逻辑代码都放这里。里面会按功能分子文件夹：

```
_Scripts/
  ├── Player/          ← 玩家控制（移动、视角、射击）
  ├── Enemy/           ← 敌人行为（AI、攻击）
  ├── Item/            ← 物品系统（拾取、弹药、血包）
  ├── UI/              ← 界面（血量、弹药显示、菜单）
  └── Manager/         ← 全局管理（游戏状态、存档）
```

### `Assets/_Prefabs/` — 预制体

把做好的游戏对象存起来，以后直接拖到场景就能用。

```
_Prefabs/
  ├── Player.prefab        ← 玩家角色
  ├── Enemy_Basic.prefab   ← 普通敌人
  ├── Pickup_Ammo.prefab   ← 弹药拾取物
  ├── Pickup_Health.prefab ← 血包拾取物
  └── Bullet.prefab        ← 子弹
```

### `Assets/_Materials/` — 材质

控制模型的外观：颜色、光滑度、金属感、发光等。

```
_Materials/
  ├── Player/          ← 玩家相关材质
  ├── Environment/     ← 场景环境材质（地面、墙壁）
  └── Enemy/           ← 敌人材质
```

### `Assets/_Textures/` — 贴图

材质的"原材料"，比如一张砖墙照片、一张铁锈纹理。

### `Assets/Scenes/` — 场景

一个场景就是一个关卡/菜单。Unity 自带一个 SampleScene。

```
Scenes/
  ├── MainMenu.unity       ← 主菜单
  ├── Level_Town.unity     ← 小镇关卡（我们的主关卡）
  └── BossRoom.unity       ← Boss 房
```

### `Assets/` 还可能出现的其他文件夹（Unity 自动生成）

| 文件夹 | 说明 |
|--------|------|
| `Resources/` | 特殊文件夹——里面的资源可以用代码动态加载 |
| `StreamingAssets/` | 不做压缩打包的原始文件（如视频、配置文件） |
| `Editor/` | 编辑器工具脚本（只在 Unity 编辑器中运行） |
| `Plugins/` | 外部插件（如 DLL 库） |
